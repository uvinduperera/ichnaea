ichnaea.async.config¶

Contains celery specific one time configuration code.

ichnaea.async.config.CELERY_QUEUES = (<unbound Queue celery_blue -> <unbound Exchange ''(direct)> -> celery_blue>, <unbound Queue celery_cell -> <unbound Exchange ''(direct)> -> celery_cell>, <unbound Queue celery_content -> <unbound Exchange ''(direct)> -> celery_content>, <unbound Queue celery_default -> <unbound Exchange ''(direct)> -> celery_default>, <unbound Queue celery_export -> <unbound Exchange ''(direct)> -> celery_export>, <unbound Queue celery_incoming -> <unbound Exchange ''(direct)> -> celery_incoming>, <unbound Queue celery_monitor -> <unbound Exchange ''(direct)> -> celery_monitor>, <unbound Queue celery_ocid -> <unbound Exchange ''(direct)> -> celery_ocid>, <unbound Queue celery_reports -> <unbound Exchange ''(direct)> -> celery_reports>, <unbound Queue celery_wifi -> <unbound Exchange ''(direct)> -> celery_wifi>)[source]¶

List of kombu.Queue instances.

ichnaea.async.config.configure_celery(celery_app)[source]¶

Configure the celery app stored in ichnaea.async.app.celery_app. This is executed both inside the master worker process and once in each forked worker process.

This parses the application ini and reads in the ichnaea.async.settings.

ichnaea.async.config.configure_data(redis_client)[source]¶

Configure fixed set of data queues.

ichnaea.async.config.init_beat(beat, celery_app)[source]¶

Configure the passed in celery beat app, usually stored in ichnaea.async.app.celery_app.

ichnaea.async.config.init_worker(celery_app, _db_rw=None, _geoip_db=None, _raven_client=None, _redis_client=None, _stats_client=None)[source]¶

Configure the passed in celery app, usually stored in ichnaea.async.app.celery_app.

Does connection, settings and queue setup. Attaches some additional functionality to the celery.Celery instance.

This is executed inside each forked worker process.

The parameters starting with an underscore are test-only hooks to provide pre-configured connection objects.

ichnaea.async.config.shutdown_worker(celery_app)[source]¶

Close outbound connections and remove custom celery_app state.

This is executed inside each forked worker process.
