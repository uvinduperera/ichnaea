ichnaea.http¶

Setup and configuration of a HTTP/S connection pool.

ichnaea.http.configure_http_session(size=20, max_retries=1, _session=None)[source]¶

Return a requests.Session object configured with a requests.adapters.HTTPAdapter (connection pool) for http and https connections.

Parameters:
  • size (int) – The connection pool and maximum size.
  • max_retries (int) – The maximum number of retries for each connection.
  • _session – Test-only hook to provide a pre-configured session.
