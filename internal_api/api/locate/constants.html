class ichnaea.api.locate.constants.DataAccuracy[source]¶

Describes the possible and actual accuracy class of a locate query.

Instances of this class can be compared based on their value or can be compared to int/float values.

These values are related to CELL_MIN_ACCURACY and CITY_RADIUS and adjustments in one need to be reflected in the other.

high = None[source]¶

High accuracy, probably Bluetooth or WiFi based.

medium = None[source]¶

Medium accuracy, probably cell based.

low = None[source]¶

Low accuracy, large cell, cell area or GeoIP.
